1. Colors

1.1. Font Colors

  • 1.1.1. Primary Font Color

    • RGB: #303030
    • SASS Variable: $color-font
    • Usage: All Text over all our light backgrounds.

  • 1.1.2. Chinese Font Color

    • RGB: #555555
    • SASS Variable: $color-font-cn
    • Usage: All Text over all our light backgrounds.

  • 1.1.3. Hint Font Color

    • RGB: #999999
    • SASS Variable: $color-hint
    • Usage: To hints and captions or every light gray font.

1.2. Links Colors

  • 1.2.1. Primary Link Color

    • RGB: #026CB6
    • SASS Variable: $color-link
    • Usage: Is the text color to all links and the background-color to all the primary action buttons. Article titles with links don't use this color.

  • 1.2.2. Hover Link Color

    • RGB: #5E95C7
    • SASS Variable: $color-link-hover
    • Usage: When a linked element is on hover state use this light blue. Is very important to article titles

1.3. Background Colors

  • 1.3.1. Primary Background Color

    • RGB: #FFFFFF
    • SASS Variable: $color-background
    • Usage: Is the body and principal background color. You have to try to avoid another backgraund colors.

  • 1.3.2. Secondary Background Color

    • RGB: #F1F1F1
    • SASS Variable: $color-background-2
    • Usage: To avoid floating sections on the white you can use this background color, a light version of gray.

  • 1.3.3. Third Background Color

    • RGB: #F7F7F7
    • SASS Variable: $color-background-3
    • Usage: Less used, if you have to make contrast with the other gray background this is a lighter version.

  • 1.3.4. Dark Background Color

    • RGB: #111111
    • SASS Variable: $color-background-dark
    • Usage: To dark backgrounds, like the top bar one, this is the color.

1.4. Border Colors

    • RGB: #EEEEEE
    • SASS Variable: $color-border
    • Usage: When an element needs a border, this have to be used solid, with 1px width and with this color.

1.5. Secondary Colors

  • 1.5.1. Red Color

    • RGB: #CC0A3C
    • SASS Variable: $color-red
    • Usage: Special to errors feedback. Use it on backgraund buttons with white text or as text color.

  • 1.5.2. Green Color

    • RGB: #00AE85
    • SASS Variable: $color-green
    • Usage: This is the success color. Like the red color you can use it as background buttons or as text color.

  • 1.5.3. Yellow Color

    • RGB: #00AE85
    • SASS Variable: $color-yellow
    • Usage: Only for text color of a warning message with white background.

  • 1.5.4. Light Yellow Color

    • RGB: #FFFFEC
    • SASS Variable: $color-yellow-light
    • Usage: Only for background color of a warning message with $font-color text

1.6. Social Colors

  • 1.6.1. Facebook Color

    • RGB: #3B5998
    • SASS Variable: $color-facebook
    • Usage: To custom facebook buttons this is the background color.

  • 1.6.2. Twitter Color

    • RGB: #55ACEE
    • SASS Variable: $color-twitter
    • Usage: To custom twitter buttons this is the background color.
