2. Fonts

{% highlight html %} {% endhighlight %} {% highlight scss %} body { font-family: $font-secondary; font-size: $font-size; } {% endhighlight %}

2.1. Occidental Fonts

2.1.1. Primary Font

This is the font used specialy on the titles. The font family used is Source Sans Pro on light (100), regular (400) and semi bold (600)

Source Sans Pro - Light (100)
Source Sans Pro - Regular (400)
Source Sans Pro - Semi Bold (600)
  • font-family: 'Source Sans Pro', 'Helvetica', sans-serif;
  • font-weight: 100, 400, 600
  • line-height: Usualy on titles it's 1.1em
  • SASS Variable: $font-primary

2.1.2. Secondary Font

This is the font used specialy on the paragraphs. The font family used is Helvetica to OSX and Arial to Windows. The strong texts use the bold weight.

Helvetica / Arial - Regular (400)
Helvetica / Arial - Bold (600)
  • font-family: 'Helvetica', 'Arial' ,sans-serif;
  • font-weight: 400, 600
  • line-height: Usualy on paragraphs it's 1.7em
  • SASS Variable: $font-secondary

2.2. Oriental Fonts

2.2.1. Chinese Font

Used on with the zh-CN site this font needs a 2em text indent on the paragraphs and by the moment it's just an sans-serif typography that change on each OS. The min-size to be read properly is 14px.

中国字体
中国字体
  • font-family: sans-serif;
  • font-weight: 400, 600
  • line-height: Usualy on paragraphs it's 1.7em
  • text-indent: On paragraphs = 2em
  • SASS Variable: $font-cn
