2. Modal sizes:

You can chose between four different widths for your modals just changing the adicional class of the 'afd-modal-content' div. These classes are 'afd-modal-size50', 'afd-modal-size60', 'afd-modal-size75', 'afd-modal-size100'.

3. Javascript:

We have instancied the object 'afdModals', and this object have the methods 'close' and 'open' (The last one allways call the first one from behind). This works as a helper to avoid being repetitive when you are writing the logic to open and close the modals.

Thus, when you are writing the logic to open a modal, you only need to call inside this scope the 'afdModals.open' method. Check the code bellow:

{% highlight js %} // With this you can open a modal when you make a click in '#my-element'. // Also, you don't need to call the 'close' method, because this is called when you call the 'open' method. $(document).ready(function() { $('#my-element').on( "click", function() { afdModals.open(); }); }); {% endhighlight %}
