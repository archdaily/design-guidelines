With those clases you can assign appearence to textareas and inputs. First, you have to wrap the input or textarea with a div with the class ".afd-textfield". Then, if you are giving style to an input, add the class ".afd-textfield__input" to it. And if you are dealing with a textarea use the class ".afd-textfield__textarea"

To add labels, put them before the input or textarea and inside the wrapper and add to it the class ".afd-textfield__label"
