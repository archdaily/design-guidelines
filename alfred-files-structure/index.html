1. Files structure

This is the structure to the styles files. There are four principal folders with files that can find on any application: Scaffold, global_components, global_layout_components and mobile. Views is the folder reserved to the styles used only on the current application. Inside the mobile folder we placed the mobile styles for all the components, including the views files.

  • .assets/
  • stylesheets/
  • └─ application.css.scss
  • └─ scaffold/
  • └─ _reset.css.scss
  • └─ _normalize.css.scss
  • └─ _variables.css.scss
  • └─ _mixins.css.scss
  • └─ _layout.css.scss
  • └─ global_components/
  • └─ _typography.css.scss
  • └─ _images.css.scss
  • └─ _buttons.css.scss
  • └─ _forms.css.scss
  • └─ _tables.css.scss
  • └─ _thumbnails.css.scss
  • └─ _articles_nav.css.scss
  • └─ _modals.css.scss
  • └─ _pagination.css.scss
  • └─ _widgets.css.scss
  • └─ global_layout_components/
  • └─ _top_bar.css.scss
  • └─ _navigation.css.scss
  • └─ _footer.css.scss
  • └─ _header.css.scss
  • └─ _mobile_navigation.css.scss
  • └─ mobile/
  • └─ global_components/
  • └─ global_layout_components/
  • └─ views/
  • └─ views/
  • └─ /* Here all the views styles for this particular application */
