1. Files structure

There are two main folders. The first one contains the framework with components that are documented in this site. Those components are generic UI units only with structural styles to be used in any place. Inside the framework lives a scaffold to setup the styles of each aplication.

The second one is a folder with specific components of our applications with complete styles. For example, the different versions of header, footer, breadcrumbs, etc.

Each of these folders have a floder with the mobile styles separated for each commponent. It keeps the files cleaner, make easyer the thebuging and allows us to import these styles at the end of all styles.

1. Files import
