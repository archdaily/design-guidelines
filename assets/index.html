1. Files structure

This is the structure to the styles files. There are four principal folders with files that can find on any application: Scaffold, global_components, global_layout_components and mobile. Views is the folder reserved to the styles used only on the current application. Inside the mobile folder we placed the mobile styles for all the components, including the views files.

  • .assets/
  • stylesheets/
  • └─ scaffold/
  • └─ _reset.css.scss
  • └─ _normalize.css.scss
  • └─ _variables.css.scss
  • └─ _mixins.css.scss
  • └─ _layout.css.scss
  • └─ global_components/
  • └─ _typography.css.scss
  • └─ _images.css.scss
  • └─ _buttons.css.scss
  • └─ _forms.css.scss
  • └─ _tables.css.scss
  • └─ _thumbnails.css.scss
  • └─ _navigation.css.scss
  • └─ _pagination.css.scss
  • └─ _widgets.css.scss
  • └─ global_layout_components/
  • └─ _top_bar.css.scss
  • └─ _navbar.css.scss
  • └─ _footer.css.scss
  • └─ _mobile_navigation.css.scss
  • └─ mobile/
  • └─ global_components/
  • └─ global_layout_components/
  • └─ views/
  • └─ views/
  • └─ /* Here all the views styles for this particular application */

1. Files importation

Here is the order to import the stylesheets inside the application.css.scss file.

{% highlight scss %} //scaffold @import "scaffold/all_files_inside_this_folder"; @import "all_library_stylesheets"; //global_components @import "global_components/all_files_inside_this_folder"; //global_layout_components @import "global_layout_components/all_files_inside_this_folder"; //views @import "views/all_files_inside_this_folder"; //mobile @import "mobile/all_files_inside_this_folder"; {% endhighlight %}
