Selects are one of the most complicated components to give styles, but here we have an easy way to standarice the appearance of them with just three steps:

  • 1. You have to wrap the select with a div with the class ".afd-selectfield"
  • 2. Add to the select the class ".afd-selectfield__select"
  • 3. After the select, inside the wrapper, add a span with the class ".afd-selectfield__caret" to emulate the caret of the native select.

The width of the select is going to fit the wide width of it container
